# ATIVIDADE

Nesta atividade, você terá a oportunidade de criar sua própria página da web usando HTML (Hypertext Markup Language), a linguagem de marcação padrão para a criação de páginas da web.

Você usará uma variedade de tags HTML para estruturar e organizar o conteúdo da sua página, incluindo cabeçalhos, parágrafos, listas, links, imagens e tabelas.

---

## Temática da Atividade

A atividade tem como temática: **"Material de Estudo Digital"**

Escolha um conteúdo estudado por você até agora na graduação, e construa uma página didática explicando aquele conteúdo, como uma nota de aula ou um material didático.

O usuário da página deve ser avaliado por um formulário ao final.
