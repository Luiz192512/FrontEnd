Repositório FrontEnd

Atividades e experimentos do 3º semestre de Engenharia de Software. Este hub reúne uma coleção de páginas estáticas, versões framework-específicas de atividades e projetos interativos com JavaScript.

📋 Visão Geral

O projeto está organizado como uma coleção de páginas estáticas e miniaplicações front-end:

  • Páginas em HTML e CSS puro
  • Versões de uma mesma atividade usando CSS puro, DaisyUI e Bootstrap
  • Projetos com JavaScript para interação, áudio e atualização dinâmica
  • Um hub principal na raiz para centralizar a navegação

🎯 Projetos do 1º Bimestre

HTMLCSS Site Colecionador

Galeria estática com uma coleção de relógios, imagens e organização visual simples. Foco em apresentação visual e estrutura de layout.

Caminho: 1-bi/site-colecionador/

Arquivos principais:

  • index.html
  • css/style.css
  • img/ (imagens dos relógios)

HTMLCSS Material de Estudo Java

Material de estudo digital com foco em Java, seções temáticas, navegação interna e formulário. Exemplo de página educacional bem estruturada.

Caminho: 1-bi/site-aprendizagem/

Arquivos principais:

  • index.html
  • css/style.css

HTMLCSS Puro Arquivo Confuso (CSS Puro)

Versão base da atividade "Arquivo Confuso da Computação" sem frameworks. Demonstra estruturação semântica com CSS próprio.

Caminho: 1-bi/site-css/

Arquivos principais:

  • index.html
  • css/style.css

DaisyUI Arquivo Confuso (DaisyUI)

Versão equivalente à atividade em Bootstrap, adaptada para DaisyUI com componentes, tema e utilitários do framework Tailwind + DaisyUI.

Caminho: 1-bi/site-daisyui/

Características:

  • Framework: DaisyUI com Tailwind CSS
  • CDN baseado
  • Sistema de componentes reutilizáveis

Bootstrap Arquivo Confuso (Bootstrap)

Versão de referência da atividade utilizando grid, utilitários e componentes prontos do Bootstrap. Demonstra uso profissional de framework CSS.

Caminho: 1-bi/site-bootstrap/

Características:

  • Framework: Bootstrap 5
  • CDN baseado
  • Layout responsivo com grid system

JavaScript Caixa Livre

Miniaplicação com JavaScript para montar compras, somar subtotais, atualizar o total e tocar efeitos sonoros. Primeira experiência com manipulação de DOM e estado.

Caminho: 1-bi/site-caixa-livre/

Funcionalidades:

  • Adição dinâmica de itens
  • Cálculo de subtotais e total
  • Efeitos sonoros (áudio)
  • Interface Bootstrap

JavaScriptCSS Temático Meu Tamagotchi Virtual

Projeto interativo com estados visuais, temporizador, reinício, áudio e easter egg escondido. Demonstra game loop e gestão de estados com JavaScript.

Caminho: 1-bi/site-bichinho/

Funcionalidades:

  • Estados do personagem (feliz, cansado, com fome)
  • Temporizador e sistema de decaimento
  • Áudio para interações
  • Easter egg escondido
  • Reinício de jogo

🚀 Como Usar

  1. Clone ou baixe o repositório
  2. Abra o arquivo index.html no navegador para acessar o hub
  3. Navegue pelos projetos através dos cards na página principal
  4. Cada projeto possui um index.html próprio na sua pasta
  5. Use a opção "Voltar ao Hub" para retornar à página principal

💡 Tecnologias Utilizadas

  • HTML5: Semântica e estruturação
  • CSS3: Estilos, animações e layouts responsivos
  • CSS Frameworks: Bootstrap, DaisyUI, Tailwind
  • JavaScript (ES6+): Interatividade e manipulação de DOM
  • Web APIs: Audio, LocalStorage, etc.

📚 Recursos de Aprendizado

Este repositório serve como portfólio de desenvolvimento front-end, cobrindo:

  • Prototipagem rápida com HTML e CSS
  • Design responsivo e mobile-first
  • Uso de frameworks CSS modernos
  • Programação orientada a eventos com JavaScript
  • Gestão de estado e manipulação de DOM
  • Integração de áudio e recursos multimídia

✨ Destaques

  • Design consistente com tema escuro e gradientes modernos
  • Navegação unificada através do hub principal
  • Projetos variados que cobrem diferentes aspectos do front-end
  • Código bem estruturado e comentado
  • Responsivo e otimizado para diferentes tamanhos de tela
